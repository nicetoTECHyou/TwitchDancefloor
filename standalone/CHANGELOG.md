# Changelog

## [0.0.1] - 2026-04-23

### Added
- Initial Release
- 14 visuelle Effekte: Laser, Scheinwerfer, Nebel, Stroboskop, Lichtkegel, Partikel, Equalizer, Tänzer, Farbflut, Spiegelkugel, Puls-Ring, Konfetti, Blitze, Rauch
- Admin Panel mit Echtzeit-Steuerung aller Effekte
- 5 Quick-Szenen: Club, Rave, Chill, Party, Blackout
- Twitch Chat Command System (anonym, kein Token nötig)
- Audio-Analyse über Web Audio API (Mikrofon, Desktop, Datei)
- WebSocket Echtzeit-Sync zwischen Overlay und Admin
- Animierte Tänzer-Silhouetten (nur links/rechts, nicht in der Mitte!)
- Standalone Server auf Port 3131
- Windows BAT-Dateien für einfachen Start
