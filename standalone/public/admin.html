💡 So bekommst du Desktop-Sound (nicht nur Mikrofon!)
Methode 1: Stereo Mix aktivieren (kostenlos, kein Extra-Software)
  1. Rechtsklick auf das Lautsprecher-Symbol in der Taskleiste → "Sounds"
  2. Tab "Aufnahme" → Rechtsklick → "Deaktivierte Geräte anzeigen"
  3. "Stereo Mix" oder "Stereomix" → Rechtsklick → "Aktivieren"
  4. Stero Mix als Standard-Aufnahmegerät setzen
  5. Dann oben im Dropdown "Stereo Mix" wählen!
Methode 2: VB-Audio Virtual Cable (kostenlos)
  1. Von vb-audio.com herunterladen und installieren
  2. Windows Sound: Wiedergabe = deine Lautsprecher, Aufnahme = CABLE Input
  3. Im Dropdown oben "CABLE Input" wählen
Methode 3: Desktop Audio Button
  1. Button klicken → Bildschirmfreigabe-Dialog öffnet sich
  2. WICHTIG: Häkchen bei "Audio teilen" / "Share system audio" setzen!
  3. Beliebigen Bildschirm wählen → Teilen
